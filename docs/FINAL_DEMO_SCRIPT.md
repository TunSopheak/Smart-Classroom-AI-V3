# Smart Classroom AI V3 - Final Demo Script

## Project Topic
Smart Classroom with AI Monitoring

## Device
Raspberry Pi 5 (2GB) / Laptop Camera for demo

## Main Problem
In a normal classroom, teachers need to check attendance manually, observe whether students are paying attention, and review classroom behavior. This system helps by combining IoT, AI monitoring, attendance, and evidence tracking.

## Demo Flow

### 1. Dashboard
Open the dashboard and explain the 3-click policy:

1. Prepare class data
2. Run AI monitoring
3. Review attendance and evidence later

### 2. Reset Demo
Click Reset Demo to prepare a clean classroom session.

Explain:
The system creates a clean demo environment with class group, subject, schedule, students, and active session.

### 3. AI Training
Open AI Training.

Explain:
The system can train student faces using camera capture or uploaded photos. Photo upload is useful when some students are absent.

### 4. AI Monitoring
Open AI Monitoring.

Click:
1. Start Camera
2. Start AI Monitoring

Explain:
The system detects students, recognizes faces, marks attendance, and monitors behavior candidates.

### 5. Face Recognition Attendance
Show the green frame box with student name.

Explain:
The system shows the student name directly on the frame box so the teacher can verify recognition immediately.

### 6. Phone-use Candidate Alert
Hold a phone clearly near the body/hand area.

Explain:
When phone-use is detected, the frame box becomes red and the system creates a behavior alert candidate.

Important:
This is not final punishment. It is a candidate alert for teacher review.

### 7. Video Evidence
Click:
1. Start Recording
2. Wait 10–15 seconds
3. Stop Recording
4. Save Clip

Explain:
The saved video includes camera view and AI frame boxes, so the teacher can review it later.

### 8. Behavior Reports
Open Behavior Reports.

Explain:
The system saves snapshot evidence when an alert happens. The teacher can review the alert later by time, session, and snapshot.

### 9. Video Records
Open Video Records.

Explain:
The system stores video evidence by session for later tracking.

### 10. Attendance
Open Attendance.

Explain:
Attendance records are saved automatically from face recognition.

## Final Explanation
This system does not replace the teacher. It supports the teacher by recording attendance, detecting behavior candidates, and saving evidence for later review.
