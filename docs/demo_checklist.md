# Smart Classroom AI V3 Demo Checklist

## Before Demo

- Start backend on port 8001
- Open Dashboard
- Click Reset Demo
- Check System Status
- Make sure Face Model is trained
- If not trained, go to AI Training and train the model

## Demo Steps

1. Dashboard
- Show total students
- Show fresh active session
- Explain Reset Demo

2. AI Training
- Select student
- Start camera
- Capture face dataset
- Train face model

3. AI Monitoring
- Start camera
- Click Analyze Once
- Show green face box
- Show blue person/body candidate box
- Show recognized student
- Show attendance marked automatically
- Show behavior person count

4. Attendance Page
- Show attendance record
- Explain Present / Late / Absent logic

5. System Status
- Show database OK
- Show model trained
- Show Raspberry Pi 5 readiness

## Important Explanation for Teacher

This system does not make final negative judgments about students.

It uses candidate review wording:

- Person candidate
- Phone-use candidate
- Looking-away candidate
- Head-down candidate
- Needs teacher review

## Demo Sentence

Smart Classroom AI V3 helps teachers reduce manual attendance work by using AI face recognition to record attendance automatically, while behavior monitoring highlights possible classroom events for teacher review.
